Biologymeetweb

Welcome to Biologymeetweb hosted on GitHub Pages.

This is my playground of JavaScript, HTML, CSS, SVG and so on. Thanks to GitHub Pages, it becomes possible to present these small tools for biologist online. All these tools are aimed to help bench scientists to make their lives easier. Currently, the following tools are available. More functionalities are coming. Please stay tuned.

  • DNAssistant -- for DNA sequence analysis.
  • AAssistant -- for protein sequence analysis.
